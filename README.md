# Webhook Relay API Go client

## Features

Currently available features:
- [x] Buckets
- [x] Inputs
- [x] Outputs
- [x] Domain reservations
- [ ] Tokens
- [ ] Functions
- [ ] Tunnels
- [ ] Regions

## Installation

You need a working [Go](https://golang.org/) environment. 

```shell
go get github.com/webhookrelay/webhookrelay-go
```
